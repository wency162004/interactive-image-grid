# Interactive Image Grid

An interactive and visually engaging image gallery built using **HTML** and **CSS Grid**.
This project demonstrates hover animations and dynamic content display using pure CSS.

##  Features

*  CSS Grid layout
*  Hover expansion effect
*  Dynamic text display on hover
*  Clean and modern UI
*  Smooth transitions


##  Tech Stack

* HTML5
* CSS3
* CSS Grid

## What I Learned

* Using CSS Grid for layouts
* Creating interactive hover effects
* Positioning elements with CSS
* Improving UI interaction without JavaScript


##  Responsiveness

This project is currently optimized for **tablet (768px) and desktop (144px) screens**.
Mobile responsiveness will be added in future updates.
